  ## 개선사항

  이 버전에서는 다음과 같은 개선사항이 적용되었습니다:

  1. **모든 페이지 구조 완성**
     - 실사 관련 페이지 완성 (실사등록, 실사조회, 실사조회 상세)
     - 조회 관련 페이지 추가 (재고/이력/거래 조회 및 상세 페이지)
     - 환경설정 관련 페이지 추가 (사용자/서버/프린터 설정 및 시스템 정보)
     - 자료전송 관련 페이지 추가 (다운로드/업로드/전송이력 조회 및 상세)

  2. **일관된 사용자 인터페이스**
     - 모든 페이지가 동일한 헤더, 콘텐츠, 푸터 구조를 공유
     - 각 페이지에서 일관된 컴포넌트와 스타일 적용

  3. **페이지 간 원활한 연결**
     - 모든 메뉴와 상세 페이지가 서로 올바르게 링크되어 있음
     - 이전 버튼을 통해 상위 메뉴로 쉽게 이동 가능

  4. **데이터 입력 및 표시 기능 개선**
     - 항목 선택, 스캔, 데이터 조회 관련 인터페이스 개선
     - 데이터 테이블 및 상세 정보 표시 개선

  5. **모바일 최적화**
     - 모든 화면이 모바일 환경에 최적화된 레이아웃 제공
     - 터치 기반 조작에 적합한 버튼 크기와 간격 적용

  ## 사용 방법

  1. 웹 브라우저에서 `index.html` 파일을 열어 메인 메뉴에 접근합니다.
  2. 각 메뉴 항목을 탭하여 다양한 기능 화면으로 이동할 수 있습니다.
  3. 화면 하단의 이전 버튼을 통해 이전 화면으로 돌아갈 수 있습니다.
  4. 각 기능별 메뉴에서 세부 화면으로 이동하여 작업을 수행할 수 있습니다.

  ## 기술 스택

  - HTML5 - 화면 구조 및 콘텐츠 정의
  - CSS3 - 반응형 레이아웃 및 스타일링
  - JavaScript - 상호작용 및 페이지 전환 효과 구현

  ## 참고사항

  - 이 프로토타입은 실제 데이터베이스와 연결되어 있지 않으며, UI/UX 디자인 및 화면 흐름 파악용으로 제작되었습니다.
  - 모든 화면은 모바일 환경과 데스크톱 환경 모두에서 사용 가능하도록 최적화되어 있습니다.
  - 실제 구현 시에는 바코드 스캔 기능, 데이터베이스 연동, 서버 통신 등이 추가로 필요합니다.

  ## 바코드 스캔 기능 사용 방법

  1. `ui/pages/input/general.html` 파일을 열어 바코드 스캔 화면에 접근합니다.
  2. 카메라 접근 권한을 허용합니다.
  3. 화면에 표시된 카메라 피드에 바코드를 맞춥니다.
  4. 바코드가 인식되면, 인식된 바코드 데이터가 화면에 표시됩니다.
